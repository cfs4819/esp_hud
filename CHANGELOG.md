# 更新日志

本文件记录项目中所有重要变更。

## 版本规则

本项目使用 `MAJOR.MINOR.PATCH` 版本号，并使用 ISO 8601 日期格式（`YYYY-MM-DD`）记录发布日期。

- `MAJOR`（`1.x.x`）：当出现不兼容的 API/协议/数据格式变更时递增。
- `MINOR`（`x.1.x`）：当新增向后兼容功能时递增。
- `PATCH`（`x.x.1`）：当进行向后兼容的问题修复、性能优化或内部重构时递增。

说明：
- 版本术语遵循 `ISO/IEC/IEEE 24765`（软件版本标识相关定义）。
- 日期表示遵循 `ISO 8601`。

## [0.2.1] - 2026-02-08

### 变更
- lvgl 画面缓存移至PSRAM。

## [0.2.0] - 2026-02-08

### 变更
- `ui_bridge` 从单队列改为 `MSG/IMG` 双队列，状态消息与图片更新解耦。
- `MSG` 路径改为覆盖写入（仅保留最新状态），`IMG` 路径保留发送语义并在满队列时淘汰最旧帧。
- 图片更新改为“只应用最新帧”，并增加上一帧延后释放，减少显示延迟与资源争用。
- 图片切换前主动失效 LVGL 图像缓存，避免同一描述符复用时读到旧图。
- LVGL 渲染参数上调：绘制缓冲从 10 行提升到 100 行，任务栈与优先级提高。
- USB 接收任务核心绑定调整到 Core 0，串口日志统一为 `Serial0`。
- Home 页面结构与布局微调：移除 `ui_Group_Content` 中间层、调整容器位置、GPS knob 默认隐藏。

### 新增
- 上位机示例增强：支持远程轨迹图拉取与更丰富的图片发送调试参数。
- README 新增线程模型与队列架构说明（MSG/IMG 分离与零拷贝说明）。

## [0.1.0] - 2026-02-08

### 新增
- 项目初始化。
- 初始 LVGL 仪表盘集成与开发板基础适配。
- USB 流接收链路，支持 `MSGF`/`IMGF` 帧路由。
- 基于 SquareLine 的 Home 页面初始版本。
- 串口发帧的上位机示例基础能力。
